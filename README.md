# Principais Comandos do Git

- git status
- git add <NomeDoArquivo>
- git commit -m'...'
- git push origin <Branch>
- git pull origin <Branch>
- git log
- git diff <NomeDoArquivo>
- git checkout <NomeDaBranch> (Acessar Branch)
- git checkout **-b** <NomeDaNovaBranch>
